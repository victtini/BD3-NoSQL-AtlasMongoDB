# BD3-NoSQL-AtlasMongoDB
Este projeto tem como objetivo criar e manipular um banco de dados NoSQL utilizando o MongoDB Atlas e o VS Code com a extensão MongoDB. As principais tarefas incluem a criação de um banco de dados chamado BD3-NoSQL-AtlasMongoDB, com a coleção bd3-nosql-atv1, onde serão realizadas operações de CRUD.
